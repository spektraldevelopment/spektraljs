getElement()

Returns an element from the DOM.

Testing Results

var
    testDivOne = Spektral.getElement("geTestOne"),
    testDivTwo = Spektral.getElement("geTestTwo"),
    testDivThree = Spektral.getElement("geTestThree");

getElementByClass()

Returns an element from the DOM based
on its class name.

Testing Results

Test Elements

div1 - class="gebc"
div2 - class="gebc"
div3- class="gebc"

Example Code

var
    testDivs = Spektral.getElementByClass("gebc"),
    testDivTwo = Spektral.getElementByClass("gebc", 1);

//Confirm testDivTwo has been properly targeted
Spektral.setStyle(testDivTwo, "border-color:green");
